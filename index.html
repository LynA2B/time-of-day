Time of Day by Evalyn Alviar

What time is it?

This page will change based on the hour of the day.

This page uses JavaScript to read the current hours and automatically updates the message, background color, and image based on the time of the day.

  • Morning (5am-12pm)
  • Afternoon (12pm-6pm)
  • Evening(6pm-11pm)
  • Night (11pm-5am)

The script uses if/else-if block, the Date object's getHours() method, and DOM methods like getElementById and querySelector to change the page dynamically. The image path is built by adding "images/" in front of filename.

The page also displays the current day of the week using getDay().
